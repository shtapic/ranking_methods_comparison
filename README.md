# 📊 Сравнение методов ранжирования (Learning to Rank)

Проект по сравнительному анализу подходов к обучению ранжированию (LTR) на примере датасета новостей (MIND).

## 📁 Файлы
* **`ranking_methods_comparison.ipynb`** — Основной ноутбук с экспериментами.
* `news.tsv` — Датасет новостей (заголовки, метаданные).
* `behaviors.tsv` — Логи действий пользователей (история кликов).
* `*.vec` — Эмбэддинги сущностей и отношений для графовых моделей.

## 🧠 Реализованные подходы
В ноутбуке сравниваются три парадигмы обучения ранжированию:

1.  **Pointwise (Поточечный)**
    *   Задача сводится к бинарной классификации (клик / не клик).
    *   Модель: **LightGBM** (LGBMClassifier).
    *   Метрика оптимизации: LogLoss.

2.  **Pairwise (Попарный)**
    *   Модель учится правильно упорядочивать пару документов (ранжировать кликнутый выше некликнутого).
    *   Модель: **RankNet** (линейная модель на разностях признаков, реализована через SGDClassifier).
    *   Принцип: $f(x_{pos} - x_{neg}) > 0$.

3.  **Listwise (Списочный)**
    *   Оптимизация метрик ранжирования списка целиком (NDCG, MAP).
    *   Модель: **LambdaMART** (LGBMRanker).
    *   Метрика: lambdarank (приближенная оптимизация NDCG).
